# داشبورد مدیریتی دو زبانه 

یک داشبورد مدرن و کامل با Next.js و React که به زبان فارسی و انگلیسی طراحی شده است.

## ویژگی‌ها

- ✅ طراحی مدرن و زیبا با Tailwind CSS
- ✅ پشتیبانی کامل از RTL (راست به چپ)
- ✅ حالت تاریک (Dark Mode)
- ✅ کامپوننت‌های قابل استفاده مجدد
- ✅ نمودارهای تعاملی با Recharts
- ✅ مدیریت کاربران
- ✅ سیستم اعلان‌ها
- ✅ گزارش‌ها و تحلیل‌ها
- ✅ تنظیمات سیستم
- ✅ طراحی واکنش‌گرا (Responsive)

## نصب و راه‌اندازی

1. نصب وابستگی‌ها:
```bash
npm install
```

2. اجرای پروژه در حالت توسعه:
```bash
npm run dev
```

3. باز کردن مرورگر در آدرس:
```
http://localhost:3000
```

## ساخت برای تولید

```bash
npm run build
npm start
```

## تکنولوژی‌های استفاده شده

- **Next.js 14** - فریمورک React
- **React 18** - کتابخانه UI
- **TypeScript** - تایپ‌اسکریپت
- **Tailwind CSS** - استایل‌دهی
- **Recharts** - نمودارها
- **Lucide React** - آیکون‌ها

## ساختار پروژه

```
dashboard/
├── app/                 # صفحات و روت‌ها
├── components/          # کامپوننت‌های React
├── lib/                # توابع کمکی
└── public/             # فایل‌های استاتیک
```

## صفحات

- `/` - داشبورد اصلی
- `/users` - مدیریت کاربران
- `/reports` - گزارش‌ها
- `/analytics` - تحلیل‌ها
- `/settings` - تنظیمات
- `/notifications` - اعلان‌ها
- `/docs` - مستندات
